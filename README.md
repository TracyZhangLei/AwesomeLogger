# AwesomeLogger
像系统自带Log一样简单的调用，日志形式更加清晰易懂，json化的数据可以直接打印出来，省去转换的麻烦，试试吧。
####用法：

compile 'cc.tracyzhang:AwesomeLogger:1.0.0'

* ALogger.d(tag,obj)
* ALogger.i(tag,obj) 
* ALogger.w(tag,obj) 
* ALogger.e(tag,obj)
